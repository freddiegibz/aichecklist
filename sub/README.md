# AI First Use Checklist

A mobile-first React checklist for older beginners who want to use ChatGPT for the first time without jargon.

## Suggested File Path

For a Next.js / Vercel project, use:

```text
app/page.tsx
```

This project already places the full interactive checklist there as a single client component.

## Local Setup

Install dependencies:

```bash
npm install
```

Start the local development server:

```bash
npm run dev
```

Then open:

```text
http://localhost:3000
```

## Deploying To Vercel

Push this folder to a GitHub repository, import the repo into Vercel, and keep the default Next.js settings.

## Notes

- No login
- No backend
- Local progress only
- Copy-to-clipboard buttons included
- PDF and 1:1 session links are placeholders:
  - `/ai-confidence-guide.pdf`
  - `/book-setup-session`
